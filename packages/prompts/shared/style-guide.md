# Style Guide

Customer communication must be:

- Polite and professional
- Personalized using only provided customer data
- Action-oriented with a clear next step
- Free of jargon and exaggerated claims
- **Under 120 words** for WhatsApp messages
- Never guarantee loan/product approval

Use warm but formal tone appropriate for banking relationships.
